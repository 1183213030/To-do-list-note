# 💌 小纸条待办 - 完整部署指南

## 📋 部署前准备

### 1. 注册微信小程序账号

1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 点击「立即注册」→ 选择「小程序」
3. 按提示完成注册（需要邮箱验证）
4. 登录后，在「设置」→「基本设置」中找到你的 **AppID**（后面会用到）

### 2. 下载微信开发者工具

1. 访问 [微信开发者工具下载页](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 根据你的系统选择对应版本下载
3. 安装并打开微信开发者工具
4. 使用微信扫码登录

---

## 🚀 正式部署步骤

### 第一步：导入项目

1. 打开微信开发者工具
2. 点击「+」或「导入项目」
3. 填写项目信息：
   - **项目名称**：小纸条待办
   - **目录**：选择本项目所在的文件夹
   - **AppID**：填入你的小程序 AppID
   - **开发模式**：小程序
   - **后端服务**：不使用云服务（我们后面手动开通）
4. 点击「导入」

### 第二步：开通云开发环境

1. 在微信开发者工具顶部菜单，点击「云开发」按钮
2. 首次使用会提示开通云开发，点击「开通」
3. 填写环境信息：
   - **环境名称**：xiaozhi（或自定义）
   - **环境 ID**：会自动生成，例如 `xiaozhi-xxxxx`
   - **配额模式**：选择「免费配额」
4. 点击「创建」，等待环境创建完成（约1-2分钟）
5. **重要：记下环境 ID**（例如 `xiaozhi-7gxxx`），后面会用到

### 第三步：配置云开发环境 ID

1. 在微信开发者工具中打开 `app.js` 文件
2. 找到第 6 行：

```javascript
env: 'your-env-id', // 替换为你的云开发环境 ID
```

3. 将 `your-env-id` 替换为你刚才记下的环境 ID，例如：

```javascript
env: 'xiaozhi-7gxxx',
```

4. 保存文件（Ctrl+S 或 Cmd+S）

### 第四步：设置云函数目录

1. 在项目根目录找到 `cloudfunctions` 文件夹
2. **右键点击** `cloudfunctions` 文件夹
3. 选择「更多设置」→「设置为云开发目录」
4. 在弹出的窗口中选择你刚创建的云环境（例如 `xiaozhi-7gxxx`）
5. 点击「确定」

### 第五步：创建数据库集合

1. 在微信开发者工具顶部点击「云开发」按钮
2. 进入云开发控制台
3. 点击左侧「数据库」
4. 点击「+」创建集合，依次创建以下 3 个集合：
   - 集合名称：`rooms`（点击确定）
   - 集合名称：`room_members`（点击确定）
   - 集合名称：`todos`（点击确定）

### 第六步：上传并部署云函数（共11个）

**重要提示**：需要逐个上传，每个云函数都要按以下步骤操作

#### 需要上传的云函数列表：
1. login
2. createRoom
3. joinRoom
4. getMyRooms
5. getRoomInfo
6. addTodo
7. getTodos
8. toggleTodo
9. updateTodo
10. deleteTodo
11. pinTodo

#### 上传步骤（每个云函数都要做）：

1. 在项目左侧目录树中，展开 `cloudfunctions` 文件夹
2. **右键点击** 云函数文件夹（例如 `login`）
3. 选择「上传并部署：云端安装依赖」
4. 等待上传完成（右下角会显示进度）
5. 看到「上传成功」提示后，继续下一个云函数

**⚠️ 注意**：
- 第一次上传可能需要 1-2 分钟，因为要安装依赖
- 如果上传失败，检查网络后重试
- 11 个云函数都要上传！

### 第七步：验证云函数部署

1. 在云开发控制台，点击左侧「云函数」
2. 查看云函数列表，应该看到 11 个云函数：
   - ✅ login
   - ✅ createRoom
   - ✅ joinRoom
   - ✅ getMyRooms
   - ✅ getRoomInfo
   - ✅ addTodo
   - ✅ getTodos
   - ✅ toggleTodo
   - ✅ updateTodo
   - ✅ deleteTodo
   - ✅ pinTodo

### 第八步：编译运行

1. 在微信开发者工具顶部，点击「编译」按钮
2. 如果没有错误，左侧会显示小程序预览界面
3. 你应该能看到 💌 小纸条 的首页

## ⚠️ 常见问题

### 1. 云开发按钮是灰色的

**原因**：项目没有关联 AppID

**解决**：
1. 点击工具栏「详情」
2. 勾选「不校验合法域名...」
3. 重新导入项目，填写正确的 AppID

### 2. 云函数上传失败

**可能原因**：
- 网络问题
- 没有设置云函数目录
- 云环境创建失败

**解决**：
1. 检查网络连接
2. 重新设置云函数目录（右键 cloudfunctions → 更多设置 → 设置为云开发目录）
3. 在云开发控制台确认环境已创建

### 3. 调用云函数报错 "cloud.init not called"

**原因**：app.js 中的环境 ID 没有配置

**解决**：
1. 打开 `app.js`
2. 确认第 6 行 `env` 填写了正确的环境 ID
3. 保存后重新编译

### 4. 数据库报错 "permission denied"

**原因**：数据库权限设置问题

**解决**：
1. 打开云开发控制台 → 数据库
2. 点击每个集合右侧的「权限设置」
3. 设置为「仅创建者可读写」

### 5. 首页一直显示"加载中"

**可能原因**：
- login 云函数没有部署
- 网络问题
- 云环境 ID 错误

**解决**：
1. 打开开发者工具「调试器」→「Console」查看错误
2. 确认 login 云函数已上传
3. 检查 app.js 中的环境 ID

### 6. 真机上无法使用

**原因**：需要上传体验版

**解决**：
1. 点击「上传」按钮
2. 在微信公众平台添加体验成员
3. 在微信中通过体验版二维码打开
